# Component Library

Detailed documentation for every hardware component used in the UniSim example projects.

- **[Raspberry Pi Pico](./components/pico.md)**: The brain of your circuit.

- **[Resistor](./components/resistor.md)**: Fundamental safety element for current limiting.
