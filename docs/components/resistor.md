# Resistor

A component that protects your circuit by slowing down electricity.

### How to Interact
- **Inspector Panel**: Click on the Resistor to open the properties panel on the right. You can type in a new resistance value (e.g., 220, 1000) or change the tolerance.
- **Setup & Real-time**: You can change these values even while the simulation is running to see how it affects brightness or safety checks.
- **What happens?**: It makes sure your LEDs don't get too much power and "burn out" virtually.

### What you see on Canvas
- **Colored Bands**: The tiny colored stripes on the resistor change based on the value you set (e.g., Red-Red-Brown for 220Ω).
- **Safety Alerts**: If you forget a resistor, the **Logs Tab** will show an error message saying "High Current Detected!"
