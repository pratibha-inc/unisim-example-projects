# Welcome to UniSim 🚀

UniSim is a state-of-the-art virtual hardware simulation environment designed for embedded systems developers, educators, and hobbyists. It allows you to build, program, and test complex electronic circuits and MicroPython/JavaScript firmware without the need for physical hardware.

### Key Features
- **Real-time Simulation**: Experience your code running on virtual microcontrollers with instant visual feedback.
- **Safety First**: Built-in validation logic prevents "virtual damage" to components (e.g., short circuits or overcurrent).
- **Dual Programming Paradigms**: Write logic using **Pure JavaScript** for high-level orchestration or **MicroPython** for near-metal firmware simulation.
- **Visual Design**: Drag and drop components, wire them up, and see them interact in a rich, 3D-accelerated interface.
- **HIL Ready**: Supports Hardware-in-the-Loop for bridging virtual simulations with real-world physical devices.
